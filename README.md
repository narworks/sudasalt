# Suda Salt — sudasalt.com

**Salt Türk Kaynak Tuzu**

Suda Salt markasının resmi web sitesi. Anadolu'nun binlerce yıllık tuz kaynaklarından, doğanın en saf hali.

## Teknoloji

- Statik HTML5 + CSS3 + Vanilla JavaScript
- AOS (Animate on Scroll) kütüphanesi
- Google Fonts (Cinzel, Playfair Display, Cormorant Garamond, Inter)
- Vercel üzerinde deploy

## Geliştirme

Projeyi lokal olarak çalıştırmak için herhangi bir statik dosya sunucusu kullanılabilir:

```bash
npx serve .
```

## Deploy

```bash
vercel --prod
```
